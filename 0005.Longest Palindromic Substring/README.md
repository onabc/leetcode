## 最长回文子串

### 描述

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

### 示例

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

### 思路

**中心扩展法：**

回文串一定是对称的，所以我们可以每次循环选择一个中心，进行左右扩展，判断左右字符是否相等即可。

![img](http://windliang.oss-cn-beijing.aliyuncs.com/5_6.jpg)

由于存在奇数的字符串和偶数的字符串，所以我们需要从一个字符开始扩展，或者从两个字符之间开始扩展，所以总共有 n + n - 1 个中心。
